# Calculadora Tkinter

Este projeto é uma calculadora simples desenvolvida com Python utilizando a biblioteca [customtkinter](https://github.com/TomSchimansky/CustomTkinter), que oferece uma interface gráfica moderna e customizável.

## Funcionalidades

- Operações básicas: soma, subtração, multiplicação e divisão
- Interface gráfica interativa
- Limpeza de entrada
- Suporte para números decimais

## Pré-requisitos

- **Python 3.7 ou superior** instalado em seu computador.
- Biblioteca `customtkinter` instalada.

## Instalação

1. **Clone o repositório**

   ```bash
   git clone https://github.com/Lucasggh/calculadora.git
   cd calculadora
   ```

2. **Instale o customtkinter**

   Você pode instalar pelo pip:

   ```bash
   pip install customtkinter
   ```

## Como executar

Na pasta do projeto, execute o arquivo principal usando Python:

```bash
python calculadora.py
```

A interface da calculadora será aberta e você poderá começar a utilizá-la!

## Uso

- Clique nos botões para digitar os números e operações.
- Pressione `=` para realizar o cálculo.
- Pressione `c` para limpar a entrada.
